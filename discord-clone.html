About Our Project

DebugThugs is a web-based application inspired by Discord. Started in early January, our goal was to deliver by April 12, 2024, a fully functional platform where users can register, log in, and manage their accounts, then join or create servers, groups, and private chats—complete with a permissions system for admins.

Project Description & Methodology

We followed Agile Scrum, with a dedicated Scrum Master facilitating weekly sprints and removing blockers. Users sign up with name, username, password, and email; returning users log in. Once inside, they can join channels, chat, invite others, or—if they’re channel admins—manage membership and messages. A friends list lets users send/accept/decline requests and have 1-on-1 conversations. Each user also has a profile where they can update personal info, bio, status, and avatar.

Tech Stack

  • React (Front-end)
  • Express & Node.js (API)
  • SQLite (User & Message DB)
  • WebSockets (Real-time Chat)
