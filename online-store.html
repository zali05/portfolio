Darkest Souls

Summary

This project was made as a halloween concept store. The emphasis was to create an online store using Java and SQL, developing a robust and scalable data schema, followed by the development of efficient real time data querying capabilities.

Core Features

  • Create Account: `accountcreation.jsp` with form‐validation and SQL transaction rollback.
  • Edit Account: `editaccount.jsp` pre‐filled form for updates, ensuring unique user IDs.
  • Product Reviews: `product.jsp` lists reviews; `reviewpage.jsp` enforces one review per user.
  • Search Bar: Instant product search from home page.
  • Menu Bar: Persistent navigation with hover feedback.
  • Shipping Confirmation: Pre‐filled address form during checkout.
  • Personalized Greeting: “Welcome, [username]” after login; “Welcome, guest” otherwise.
  • Category Search: Partial‐match lookup for product categories.

Walkthrough

  1. Home page with menu and search bar to find products.
  2. Add items to cart and proceed to checkout; authenticate credentials.
  3. Confirm or edit shipping address on a pre‐filled form.
  4. Handle out‐of‐stock errors gracefully.
  5. Browse by category via hint‐based search.
  6. Create and edit user accounts with live validation.
  7. Submit and view product reviews; admins can monitor all orders.

Tech Stack

  • Java & JSP (accountcreation.jsp, editaccount.jsp, product.jsp, reviewpage.jsp)
  • Servlets & JDBC for SQL transactions
  • HTML, CSS & JavaScript for front‐end
  • Relational Database (MySQL or SQLite)
  • Tomcat (or equivalent servlet container)
