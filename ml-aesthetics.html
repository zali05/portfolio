ML Aesthetic Image Analysis

Overview

Our system is a web-based user survey platform designed to collect responses on image aesthetics and provide powerful tools for survey management, data analysis, and visualization. It supports two user groups—end users who participate in the aesthetic survey and administrators who configure surveys and export analytics. Surveys are driven by JSON configuration files, and the interface is bilingual (English/French). :contentReference[oaicite:0]{index=0}​:contentReference[oaicite:1]{index=1}

System Architecture

  • Frontend: React.js for a fast, component-based UI.
  • Backend: Node.js + Express.js handling API logic and data workflows.
  • Database: SQLite for lightweight, file-based storage of users, responses, and configs.

Aesthetics Prediction Model

We predict continuous aesthetic scores (1–10) by combining: (1) ResNet50 visual embeddings → PCA (retain 80% variance), (2) Handcrafted metrics (brightness, contrast, sharpness, symmetry, etc.), (3) ViT-generated semantic labels (one-hot encoded). These feature sets are concatenated and fed into a Random Forest Regressor for final score prediction. :contentReference[oaicite:12]{index=12}​:contentReference[oaicite:13]{index=13}

Image Generation Model

We use Stable Diffusion enhanced with ControlNet to generate high-fidelity images guided by structural hints. This combo outperforms BigGAN in detail, texture, and inference speed (~2.8s vs. ~10.7s per image). :contentReference[oaicite:14]{index=14}​:contentReference[oaicite:15]{index=15}

Evaluation Approach

Since aesthetics is subjective, we ran a user study (10 participants, 1–5 scale) and tracked computational metrics (latency, throughput, memory). Stable Diffusion images averaged 4.02/5 versus BigGAN’s 2.1/5, confirming user preference for our generation pipeline. :contentReference[oaicite:16]{index=16}​:contentReference[oaicite:17]{index=17}

On the ML model side, we measured MSE (0.5741) and R² (–0.0182) on a hold-out AVA dataset split, with a pseudo-accuracy of ~91.6%. :contentReference[oaicite:18]{index=18}​:contentReference[oaicite:19]{index=19}

Conclusion

This platform not only captures human aesthetic judgments via a seamless, multilingual survey but also leverages them to train and evaluate ML models and generation pipelines. Our modular design—React frontend, Express/SQLite backend, and Python-based ML—ensures portability and extensibility for future research and feature expansion.
